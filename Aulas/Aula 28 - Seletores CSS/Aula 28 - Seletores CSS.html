Aula 28 - Seletores CSS

Nesta aula aprendi a utilizar o Internal Style Sheet dentro da tag < head >, possibilitando atribuir um mesmo estilo para várias tags iguais dentro do < body >.

Se necessário modificar todos os parágrafos para vermelho, por exemplo, basta criar a tag < style type="text/css" > dentro do < head >.


< head >
< title >Internal Style Sheet< /title >
< meta charset="utf-8" >

< style type="text/css" >

< /style >

< /head >


Colocar o nome da tag que deseja estilizar e definir a modificação entre colchetes (similar ao C).

< head >
< title >Internal Style Sheet< /title >
< meta charset="utf-8" >

< style type="text/css" >

p
{
color: red;
}

< /style >
< /head >
