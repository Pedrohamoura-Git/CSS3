Criando Barra de Navegação Lateral

Nesta aula aprendi como marcar na barra lateral a seção em que o usuário se encontra, criando classes que definem um estilo diferente para a opção da barra de navegação correspondente à página atual.

< ul #Selected_Links >

< li>
< a href="aula 56.html">Home
< /li>

< li>
< a href="quemsomos.html">Quem somos
< /li>

< li class="noticias">
< a href="noticias.html">Notícias
< /li>

< li>
< a href="contatos.html">Contatos
< /li>

< /ul>
