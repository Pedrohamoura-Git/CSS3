Aula 32 - Cores & Fontes

Nesta aula, aprendi a utilizar cores através de seus códigos hexadecimais.


#HeaderBorder
{
border: solid;
border-color: purple green #088A08 #6A0888;
}

.red
{
color: #FF0040;
}

Links de referência: Paleta de cores

Aprendi também a formatar um texto.

.textformat
{
font-size: 18px;
font-family: "Times New Roman", Georgia, Garamond, serif;
}

font-family define quais fontes serão utilizadas, por ordem de preferência, pelo navegador.
