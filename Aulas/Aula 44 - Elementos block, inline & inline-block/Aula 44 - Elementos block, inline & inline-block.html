Aula 44 - Elementos block, inline & inline-block

Nesta aula aprendi a utilizar o modificador display para alterar o comportamento dos contêiners dentro da página.

O tipo block ocupa toda uma linha da tela.

.border-format1
{

border-style: inset;
border-color: red;
display: block;
margin: 10px;

/*
display: inline;
display: inline-block;
*/

}

O tipo inline não ocupa toda uma linha da tela, permitindo que vários elementos inblock sejam organizados um ao lado do outro.

.border-format
{

border-style: inset;
border-color: red;
display: inline;
margin: 10px;

/*
display: block;
display: inline-block;
*/

}

O inline-block prioriza organizar os contêiners um ao lado do outro (inline) enquanto houver espaço na página. Do contrário, ele salta uma linha (block) e organiza lado a lado novamente (inline).

.border-format
{

border-style: inset;
border-color: red;
display: inline-block;
margin: 10px;

/*
display: block;
display: inline;
*/

}
