Padding modifica os espaçamentos internos de uma caixa.
Este elemento pode receber até 4 parâmentros, representando os 4 espaçamentos internos - começando pelo topo e continuando em sentido horário.

Ex:

.tableformat
{
border: 1px solid red;
padding: 7px 14px 21px 42px;

/*
**padding-top: 7px;
**padding-right: 14px;
**padding-bottom: 21px;
**padding-left: 42px;
*/

}

Margin altera o espaçamento exterior da caixa.

.tablemargin
{
border: 1px solid blue;
margin: 7px 14px 21px 42px;

/*
**margin-top: 7px;
**margin-right: 14px;
**margin-bottom: 21px;
**margin-left: 42px;
*/

}
