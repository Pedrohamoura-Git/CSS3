Aula 43 - Elementos flutuantes na prática

Nesta aula aprendi a formatar as caixas usando o clear para evitar que os elementos dentro do fluxo flutuante interfiram em outras tags.


Exemplo com o clear:

{
clear: both;

/*
clear: left;
clear: right;
clear: none;
*/
}
