Aula 48 - Formatando links

Os links possuem alguns estados que devem ser levados em consideração quando for necessários formatá-los.

É importante que a ordem de definição abaixo seja mantida para que os códigos funcionem corretamente.

Não visitado:

a:link
{
color: orange;
}

a:active
