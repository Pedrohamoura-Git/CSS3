Aula 35 - Cor & Imagem de Fundo

Nesta aula aprendi a modificar a cor de fundo, a como acrescentar uma imagem e como manipulá-la dentro da página.


Definindo a repetição da imagem de fundo:

.background-repeat
{
background-repeat: repeat;

/*
repeat-x;
repeat-y;
no-repeat;
*/
}

Resultado: Aula 35 - Background-repeat
Links de referência: CSS Backgrounds

Definindo a posição da imagem de fundo:

.background-position
{
background-repeat: no-repeat;
background-attachment: fixed;
background-position: bottom left;

/*
background-position: eixo y (top, center, bottom) eixo x (left, center, right)
*/
}
Resultado: Aula 35 - Background-position
Links de referência: Idem

Definindo a resposta da imagem à rolagem da página:

.background-attachment
{
background-attachment: fixed;

/*
scroll; (Acompanha o scroll da página)
*/
}
