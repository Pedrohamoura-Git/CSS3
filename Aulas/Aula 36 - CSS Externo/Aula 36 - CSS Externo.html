Aula 36 - CSS Externo

Nesta aula aprendi a criar e priorizar a estilização da página a partir de um arquivo externo (CSS Externo).

Para utilzar um css externo é necessário usar tag < link > no head.
Em seguida, também é preciso definir o elemento rel(relation) e, neste caso, atribuí-lo com "Stylesheet".
Após, vêm o type (tipo) do arquivo a ser linkado: "text/css".
Por fim, href (Hipertext REFerence) do arquivo css em questão.


< head >

< title >Aula 36 - CSS Externo< /title >
< meta charset="utf-8" >

< link rel="stylesheet" type="text/css" href="Aula 36 - Style.css" >

< /head >

Resultado: O layout atual da página.
