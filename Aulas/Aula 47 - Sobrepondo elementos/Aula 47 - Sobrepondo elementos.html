Aula 47 - Sobrepondo elementos

Nesta aula aprendi a sobrepor elementos usando z-index. Este modificador permite numerar, em ordem de prioriadade crescente, quais elementos devem aparecer primeiro na tela.

Neste caso, a caixa 2 têm a maior prioridade. Seguida da caixa 1 e, por fim, a caixa 3.

.caixa1
{
background-color: red;
border: solid white;
height: 131px;
width: 210px;

position: absolute;
top: 210px;
right: 500px;
z-index: 1;

text-align: center;
padding-top: 100px;
color: white;
}


.caixa2
{
background-color: green;
border: solid white;
height: 131px;
width: 210px;

position: absolute;
top: 380px;
right: 618px;
z-index: 2;

text-align: center;
padding-top: 100px;
color: white;
}


.caixa3
{
background-color: blue;
border: solid white;
height: 131px;
width: 210px;

position: absolute;
top: 380px;
right: 400px;
z-index: 0;

text-align: center;
padding-top: 100px;
color: white;
}
