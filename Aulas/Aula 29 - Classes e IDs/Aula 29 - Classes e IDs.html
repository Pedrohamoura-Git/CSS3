Aula 29 - Classes e IDs

Classes

As classes são utilizadas quando necessário aplicar algum tipo de modificação em uma tag dentro do < body >; a mesma classe pode ser aplicada em mais de uma tag e ela pode conter vários modificadores.

Uma classe é definida com um "." seguido de seu nome.
É recomendado usar um nome que descreva sua funcionalidade geral ao invés de sua descrição exata. Do contrário, se algum elemento for modificado posteriormente dentro da classe, seu nome pode se tornar obsoleto.
Ex: .HeaderStyle é melhor que .HeaderColorBlue ou .HeaderColor

Sintaxe:
.HeaderStyle
{
color: blue;
}

IDs

Uma ID, por sua vez, também serve para estilizar uma tag. Entretanto, uma ID não pode ser utilizada mais do que uma vez dentro de um projeto - da mesma forma que não existem 2 pessoas com o mesmo CPF.
Uma classe é definida com "#" seguido de seu nome. As recomendações de nomeclatura de classe também se aplicam aos IDs.

Sintaxe:
#FirstHeaderStyle
{
color: green;
}
